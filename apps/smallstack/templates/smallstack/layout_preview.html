Sidebar Collapsed

config/settings/base.py 
SMALLSTACK_SIDEBAR_DEFAULT = "closed"
SMALLSTACK_TOPBAR_NAV_ALWAYS = True
views.py 
def get(self, request, *args, **kwargs):
    request._smallstack_sidebar_state = "closed"
    return super().get(request, *args, **kwargs)
Sidebar starts closed but users can toggle it open. Topbar nav fills in automatically.

Sidebar State Precedence

#SourceExample
1Template block{% templatetag openblock %} block sidebar_state {% templatetag closeblock %}disabled{% templatetag openblock %} endblock {% templatetag closeblock %}
2View contextrequest._smallstack_sidebar_state = "closed"
3User preferencelocalStorage (set by toggle button click)
4Global settingSMALLSTACK_SIDEBAR_DEFAULT = "open"
When set via template block or view context, state is forced — JavaScript won't override it with localStorage.
